# 관리자 등록 및 수정, 삭제

## 관리자 등록 및 수정, 삭제

#### ❶ appspeed내의 booking 페이지([https://service.appspeed.net/apps/booking](https://service.appspeed.net/apps/booking))에 **접속**합니다.

#### ❷ 나타난 로그인 창에 초기설정으로 발급된 아이디와 비밀번호를 입력하고 **로그인**합니다.

![image](https://user-images.githubusercontent.com/72512101/127766654-a0e6f986-357b-4661-bcdb-8355719fe29f.png)

#### ❸ 좌측 하단의 `관리자 관리`를 클릭합니다.

![image](https://user-images.githubusercontent.com/72512101/127766712-39016b79-16d6-468c-b464-81a0085f4f06.png)

#### ❹ 표 좌측 상단에 있는 `새 관리자 등록` 버튼을 클릭합니다.

![image](https://user-images.githubusercontent.com/72512101/127766743-c7bd8654-dca1-4f9f-8ba4-91b62aebc810.png)

#### ❺ 해당 버튼을 클릭하면 나오는 모달창에 항목을 모두 입력하고 `OK`를 누릅니다.

![image](https://user-images.githubusercontent.com/72512101/127766755-95d48503-ca47-4afb-a8e0-a304bd6ce2cd.png)

#### ❻ 관리자를 수정, 암호변경, 삭제를 원하는 경우 작업항목에 있는 `…` 버튼을 눌러서 원하는 작업을 선택합니다.

![image](https://user-images.githubusercontent.com/72512101/127766780-c893ea53-0782-4872-8414-b5d83b7e2eff.png)
