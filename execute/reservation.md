# 공휴일 관리, 승인 관리, 예약 대리 신청

## **공휴일 관리**

#### ❶ appspeed내의 booking 페이지([https://service.appspeed.net/apps/booking](https://service.appspeed.net/apps/booking))에 **로그인하여 접속**합니다.

#### ❷ 좌측의 `공휴일관리` 메뉴를 클릭합니다.

![image](https://user-images.githubusercontent.com/72512101/127767278-4262dd54-c6b0-4dea-b90c-358f49d76269.png)

#### ❸ 원하는 월을 **달력 상단의 버튼**으로 이동한다.

![image](https://user-images.githubusercontent.com/72512101/127767297-aa0461fd-b8ea-49a7-8cf4-9133329d70f1.png)

#### ❹ **전체** 혹은 **기관**을 선택하여 **사유란**에 텍스트를 적고 `저장` 버튼을 누르면 해당 날짜가 휴일로 설정되고 예약 페이지의 해당 날짜가 선택 불가로 설정됩니다.

![image](https://user-images.githubusercontent.com/72512101/127767337-aec95c62-d984-4467-9a55-f31cacbcec16.png)

## **승인 관리**

#### ❶ appspeed내의 booking 페이지([https://service.appspeed.net/apps/booking](https://service.appspeed.net/apps/booking))에 **로그인하여 접속**합니다.

#### ❷ 좌측의 `승인관리` 메뉴를 클릭합니다.

![image](https://user-images.githubusercontent.com/72512101/127767380-3397c5ed-e1fe-479a-b62b-1eff68f44594.png)

#### ❸ 표 내부의 요약된 데이터를 선택하면 하단에 더 자세한 신청서를 볼 수 있습니다.

![image](https://user-images.githubusercontent.com/72512101/127767397-5231b6d5-2a15-4636-84e9-9ccb9b913e5e.png)

![image](https://user-images.githubusercontent.com/72512101/127767406-e0ddf392-1193-4e70-bf75-b9056102b112.png)

#### ❹ 신청 내역을 보고 **승인/반려** 여부를 선택하여 표 좌측 상단의 `저장` 버튼을 누릅니다.

![image](https://user-images.githubusercontent.com/72512101/127767422-cd9dac63-8fd5-447a-afd9-53ca4d5106b6.png)

## **예약 대리 신청**

### ❶ appspeed내의 booking 페이지([https://service.appspeed.net/apps/booking](https://service.appspeed.net/apps/booking))에 **로그인하여 접속**합니다.

#### ❷ 좌측의 `예약대리신청` 메뉴를 클릭합니다.

![image](https://user-images.githubusercontent.com/72512101/127767502-2bc27f7a-b670-4e2b-aee1-88e8fb847ad9.png)

#### ❸ 표 내부의 항목을 모두 채운 뒤 `저장` 버튼을 누른다. 이 메뉴를 이용하여 예약 신청 시, 별도의 승인은 필요 없이 확정된다.

![image](https://user-images.githubusercontent.com/72512101/127767526-26d13446-f2d7-4025-a18c-128e70d5694d.png)
